## 核心模式：先考后学

> 不是你学什么我就教什么，而是你不会什么我就教你什么

| 对比项 | 传统学习平台 | 本平台 |
|--------|------------|-------|
| 起点 | 从第一课开始学 | 先诊断，从你不会的地方开始 |
| 课程 | 所有人一样 | 千人千面，动态定制 |
| 效率 | 60% 时间在学已会的 | 100% 时间在攻克盲区 |

## 核心流程

```
诊断面试 → 精准备课 → 循环学习 → 验收面试
```

### 1. 诊断面试（三遍扫描）
- 广度扫描（10min）— 快速覆盖知识模块
- 深度追问（15min）— 从红区深挖
- 综合题（10min）— 跨知识点应用

### 2. Teacher 精准备课
| 诊断结果 | 策略 |
|---------|------|
| ✅ 熟练 | 直接跳过，出进阶题挑战 |
| ⚠️ 模糊 | 快速回顾 + 重点练习 |
| ❌ 不会 | 从基础到进阶系统讲解 |

### 3. 三层循环
- 大循环（课程级，4-6周）
- 中循环（章节级，1周）
- 小循环（知识点级，10-20min）
